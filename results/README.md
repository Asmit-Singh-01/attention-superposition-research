# Results

This directory contains experimental results.

Results will be added only after experiments are executed.

No experimental claim will be made without supporting evidence.

---

## Baseline Data

The `baseline/` directory will contain extracted representations
from the initial Transformer experiments.

Expected data:

- Query (Q) representations
- Key (K) representations
- Value (V) representations
- Attention scores
- Attention weights

These files will be used in subsequent representation geometry
and sparse decomposition experiments.
